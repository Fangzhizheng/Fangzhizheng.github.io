如何查看npm -g全局安装的路径

十一月 11, 2020

1. 可通过命令 npm config get prefix 查看 npm 安装路径（npm config set prefix 可设置 npm 安装路径）；

2.打开该路径，在 npm 目录下有个 node_modules 文件夹，全局安装的所有模块都在该文件夹中。
